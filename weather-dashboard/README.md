# Weather Dashboard 🌤️

## ✨ Demo Live
[Weather Dashboard](https://RobertoOseguera.github.io/weather-dashboard)

## Features
✅ **Tiempo real** OpenWeatherMap API  
✅ **Pronóstico 7 días**  
✅ **Geolocalización** auto-detect  
✅ **UV Index** + Sensación térmica  
✅ **Responsive** mobile-first  
✅ **UX moderna** glassmorphism + icons  

## Tech Stack
```
HTML5 | CSS3 | Vanilla JavaScript
OpenWeatherMap API | Async/Await
Geolocation API | Responsive Grid
Modern CSS (Gradients, Backdrop-filter)
```

## API Setup
1. Regístrate gratis [OpenWeatherMap](https://openweathermap.org/api)
2. Copia API key gratuita (3 calls/min)
3. Reemplaza `API_KEY` en script.js

## Deploy
```
git init
git add .
git commit -m "Weather Dashboard"
gh repo create weather-dashboard --public --push
```
