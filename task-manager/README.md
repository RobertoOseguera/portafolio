# Task Manager Pro 🚀

## ✨ Demo Live
[Task Manager](https://RobertoOseguera.github.io/task-manager)

## Features
✅ **CRUD completo** - Create, Read, Update, Delete  
✅ **Drag & Drop** - Reordenar tareas  
✅ **LocalStorage** - Persistencia automática  
✅ **Filtros** - Todas/Pendientes/Completadas  
✅ **Stats reales** - Contadores actualizados  
✅ **Export/Import JSON** - Backup fácil  
✅ **Responsive** - Mobile-first  
✅ **Animaciones** - UX smooth  

## Tech Stack
```
HTML5 | CSS3 | Vanilla JavaScript
LocalStorage | Drag & Drop API
Responsive Design | Modern CSS
```

## Deploy GitHub Pages
```
git init
git add .
git commit -m "Initial Task Manager"
gh repo create task-manager --public --push
```

## Uso
1. **Nueva tarea**: Escribe + Enter  
2. **Completar**: Checkbox  
3. **Reordenar**: Drag & drop  
4. **Export**: JSON backup  

⭐ **Perfecto para portafolio junior** - demuestra DOM, events, storage, UX.
