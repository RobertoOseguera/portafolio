# Expense Tracker Pro 💰

## ✨ Demo Live
[Expense Tracker](https://RobertoOseguera.github.io/expense-tracker)

## Features
✅ **Balance real-time** Ingresos vs Gastos  
✅ **Charts interactivos** Chart.js Pie/Doughnut  
✅ **6 Categorias** con icons personalizados  
✅ **CRUD completo** Add/Edit/Delete  
✅ **LocalStorage** Persistencia automática  
✅ **Export CSV** Excel compatible  
✅ **Responsive** Mobile + Desktop  
✅ **UX moderna** Glassmorphism + Animations  

## Tech Stack
```
HTML5 | CSS3 | Vanilla JavaScript
Chart.js | LocalStorage
Grid/Flexbox | Modern CSS
Responsive Design
```

## Uso
1. **Agregar**: Amount + Description + Category + Type  
2. **Visualizar**: Charts actualizados real-time  
3. **Export**: CSV para Excel/Google Sheets  
4. **Reset**: Botón limpiar todo
